To run this project using Postman use the following link and access it using POST method to get By Category and add a JSON Object under the Body field such as the one shown below:
<img width="982" alt="image" src="https://user-images.githubusercontent.com/67845783/133920633-d94f83b0-56f1-4b48-a558-00ea6b10e4f8.png">

You can try adding as many JSON Objects as you want.

To access a specific category, add the name of the category in the url such as http://localhost:8080/products/Pants where the category is Pants as shown in the image below.
<img width="968" alt="image" src="https://user-images.githubusercontent.com/67845783/133920700-3fe94c63-c5ad-4fe6-a73b-c1c721853a21.png">
The API willl be able to search the category and display the JSON Objects based on the category entered in url.

Then, use the link to server http://localhost:8080/h2/ using Postman or any browser.
